# notas-atletas

> Desafio do curso de Lógica de Programação — DevStart / SENAI

Descrição
- Projeto simples em JavaScript que faz parte do desafio do curso de lógica de programação do DevStart/SENAI. O arquivo principal é `notas-atletas.js` e contém a lógica do exercício.

Tecnologias
- JavaScript (Node.js)

Pré-requisitos
- Ter o Node.js instalado (versão 12+ recomendada).

Como executar
1. Abra um terminal na pasta do repositório.
2. Execute:

```bash
node notas-atletas.js
```

Créditos
- Desafio do curso de Lógica de Programação — DevStart / SENAI.
